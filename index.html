Il mio primo articolo

Oggi abbiamo creato una semplice pagina web usando GitHub. L’obiettivo dell’esercitazione è capire come si pubblica un contenuto online.

Non serve essere programmatori: basta conoscere alcuni passaggi fondamentali e usare gli strumenti giusti.

28 maggio 2026

Cosa ho imparato

Ho imparato che una pagina web può essere composta anche da un solo file HTML. GitHub Pages permette di pubblicarla gratuitamente online.

28 maggio 2026

Perché è utile per un facilitatore digitale

Un facilitatore digitale aiuta le persone a usare meglio gli strumenti online. Saper creare una piccola pagina web aiuta a comprendere meglio come funziona la pubblicazione dei contenuti su Internet.
